## Naming conventions

1. Agent projects: `FinWise.<AgentName>Agent.csproj`
2. Shared libraries: `FinWise.Shared.<Area>.csproj`
3. Test projects: `FinWise.<Component>.Tests.csproj`
4. Function endpoint files: `<AgentName>Functions.cs`
5. Orchestration/services: `<Domain>Orchestrator.cs` or `<Domain>Service.cs`
6. State persistence: `<Domain>StateHandler.cs`

## Notes

1. `FinWise.Shared.Contracts` was removed. Contracts are now consolidated under `Shared\FinWise.Shared.Core`.
2. `Infrastructure/` is not currently present at repo root.
3. Some test folders may exist as placeholders/artifacts and are not necessarily active solution projects.
